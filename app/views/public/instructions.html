To start visit www.comsatracker.com in your mobile phone or tablet browser. We support iOS, Android and Windows devices.

Check In



  • Search for swimmer by last name.
  • Click the CHECK IN button to check in the swimmer. If no phone number is present please see below.
  • Click the ADD PHONE button, this will open a pop up where you can enter in the swimmers 10-digit phone number (with no dashes). Then click CHECK IN.

Check Out



  • Search for swimmer by last name or scroll down to find the correct swimmer.
  • Click the CHECK OUT button to check out the swimmer.
  • If you need to call the swimmer simply click CALL.

Out of State



  • Search for swimmer by last name.
  • Click the ADD button to add the swimmer so they can be searched on the Check In page.
  • Follow the steps from the Check In section.
